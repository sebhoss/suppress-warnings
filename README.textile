h1. Suppress-Warnings "!https://secure.travis-ci.org/sebhoss/suppress-warnings.png!":http://travis-ci.org/sebhoss/suppress-warnings "!https://www.ohloh.net/p/suppress-warnings/widgets/project_thin_badge.gif!":https://www.ohloh.net/p/suppress-warnings/

p. Constants for @SuppressWarnings annotations.


h2. Usage

p. Simply reference this module as a dependency to your POM:

bc. 
<dependency>
  <groupId>com.github.sebhoss</groupId>
  <artifactId>suppress-warnings</artifactId>
  <version>[1.0.0,2.0.0)</version>
</dependency>
